#MakersBnb
##Potential User Stories

```
As a User,
I want to receive an email when
* I sign up
* I create a space
* I update a space
* Someone requests to book my space
```
```
As a User(Renter),
I want to receive an email when
* My request for a space is confirmed
* My request is confirmed
* My request is denied
```
```
As a User,
I want to receive a text when
* Someone requests to book my space
* My request is confirmed
* My request is denied
```
```
As a User,
I would like a chat functionality once a space has been book in order for both
parties to communicate
```
```
As a Dev,
I'd like users to have basic pay implementation through Stripe
```
